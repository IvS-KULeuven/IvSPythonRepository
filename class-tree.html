Class Hierarchy

  • ivs.aux.termtools.CallInstruct: Generate a callable function on-the-fly.
  • ivs.aux.decorators.LogPrinter: LogPrinter class which serves to emulates a file object and logs whatever it gets sent to a Logger object at the INFO level.
  • ivs.units.conversions.NonLinearConverter: Base class for nonlinear conversions
  • ivs.stellar_evolution.puls.StellarModel: Class facilitating reading in of stellar models from different codes, and calculating frequencies with different codes,
  • ivs.aux.xmlparser.XMLParser: Convert a XML page to a dictionary
  • object: The most base type
    • ivs.observations.visibility.Ephemeris
    • logging.Filterer: A base class for loggers and handlers which allows them to share common code.
    • ivs.sigproc.fit.Function: Class to define a function with associated parameters.
    • ivs.catalogs.hermes.HermesCCF: Object that holds all information stored in the *_AllCCF.fits files written by hermesVR.
    • ivs.statistics.linearregression.LinearFit: A class that allows to easily extract the fit coefficients, covariance matrix, predictions, residuals, etc.
    • ivs.statistics.linearregression.LinearModel: A linear model class
    • ivs.aux.termtools.MemoryMonitor
    • ivs.sigproc.fit.Minimizer: A minimizer class on the lmfit Python package, which provides a simple, flexible interface to non-linear least-squares optimization, or curve fitting.
    • ivs.sigproc.fit.Model: Class to create a model using different Functions each with their associated parameters.
    • ivs.aux.progressMeter.ProgressMeter
    • ivs.sed.builder.SED: Class that facilitates the use of the ivs.sed module.
    • ivs.sed.builder.SampleSEDs: Class representing a list of SEDs.
    • ivs.units.conversions.Unit: Class to calculate with numbers (and uncertainties) containing units.
    • ivs.aux.termtools.Wrapper: Wrap the module so that the getattr function can be redefined.
    • ivs.aux.decorators.countcalls: Keeps track of the number of times a function is called.
    • dict: dict() -> new empty dictionary dict(mapping) -> new dictionary initialized from a mapping object's (key, value) pairs dict(iterable) -> new dictionary initialized as if via: d = {} for k, v in iterable: d[k] = v dict(**kwargs) -> new dictionary initialized with the name=value pairs in the keyword argument list.
