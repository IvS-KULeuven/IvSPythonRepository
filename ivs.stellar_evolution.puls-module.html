repartition(xx, n_new=None, smooth=9, sigma=None, full_output=False)

source code 

Repartition and increase length of a vector.

The new partition of the vector is more smoothed.

Keyword smooth sets the smoothness (the higher, the smoother). The keyword sigma can be used to give additional weights to regions in the star.

The start and end of the new xx will be exactly the same as the original.

Parameters:
  • xx (ndarray) - original x vector
  • n_new (None/int) - length of the new repartitioned vector (defaults to length of original vector)
  • smooth (int) - smoothness parameter
  • sigma (ndarray) - weights for repartitioning
  • full_output (bool) - if True, function returns new x and weights
Returns: ndarray
repartitioned vector

remesh(starl, x='x', n_new=None, type_mode='g', smooth_mesh=9, smooth_weights=0.005, smooth_profile=0.002)

source code 

Improve sampling of a local star record array.

The are the actions that are taken:

  1. A new x-axis is generated, locally smoothed (smooth_mesh, see repartition and possibly locally adapted for the type of modes (type_mode, see generate_newx). The latter uses the Brunt-A, which can be smoothed with smooth_weights (see smooth_bruntA in generate_newx).
  2. All quantities defined in the local star record array are interpolated on the new mesh, with the possibility to locally smoothen the profiles, with a smoothing factor that is proportional to the logarithm of the brunt A but inverse propertional to the distance within the star (smooth_profile=0.002). The interpolation method is the local polynomial interpolation of Marc-Antoine Dupret.
